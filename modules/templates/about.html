About the project

This website is created as a university cycle of home tasks. In order to perform this job I used a variety of python libraries and engaged my knowledge of object-oriented programming, knowledge of working with API and designing abstract data types. As well, I needed to explore the power of Dash framework and its capabilities to create interactive apps using Reach js framework under its hood. Except theses points, the simple designing of the wrapper site was done in responsive and adaptive way using Bootstrap Web UI Framework, that means the site will have an appropriate layout both in large and small devices.
